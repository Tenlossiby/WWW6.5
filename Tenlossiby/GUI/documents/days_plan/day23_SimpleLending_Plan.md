# 📋 Day 23 - SimpleLending 借贷平台 详细规划
## 一、合约核心功能分析
基于 day23-SimpleLending.sol ，合约包含以下核心功能：

功能模块 函数 说明 存款 deposit() 用户存入ETH获取利息收益 取款 withdraw(amount) 用户取回存款 抵押品 depositCollateral() 存入ETH作为借款担保 取抵押 withdrawCollateral(amount) 取回抵押品（需满足抵押率） 借款 borrow(amount) 根据抵押品借出ETH 还款 repay() 偿还借款（本金+利息） 利息计算 calculateInterestAccrued(user) 计算累积债务
